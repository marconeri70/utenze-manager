# Gestione Utenze 2.2

Novità:

- tasto **Modifica scheda** in ogni fattura;
- modifica successiva di fornitore, tipo, scadenza, importo, numero fattura,
  periodo, POD, PDR, codice cliente, indirizzo e consumi;
- possibilità di sostituire o aggiungere il PDF durante la modifica;
- tasto **Crea rateizzazione** anche dopo aver già salvato la fattura;
- generazione automatica iniziale di 2 rate, sempre modificabili;
- conservazione delle rate e delle ricevute quando il piano non viene cambiato;
- avviso prima di sostituire un piano rate contenente pagamenti o ricevute;
- correzione della visualizzazione duplicata dell'unità di consumo;
- cache PWA aggiornata alla versione 2.2.

Carica sul repository tutti i file contenuti nello ZIP, mantenendo i nomi esatti.

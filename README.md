# Gestione Utenze 2.3 — OCR automatico

Novità:

- legge normalmente i PDF che contengono testo digitale;
- se il PDF è una scansione, attiva automaticamente il riconoscimento OCR;
- converte le pagine PDF in immagini e le legge in italiano;
- mostra pagina, fase e percentuale di avanzamento;
- invia il testo riconosciuto al Worker AI per compilare i campi;
- mantiene l'estrazione locale come recupero di emergenza;
- analizza fino a 10 pagine per evitare blocchi di memoria sui telefoni;
- mantiene modifica fatture, rateizzazione successiva, ricevute e sincronizzazione Cloudflare.

Il primo utilizzo OCR può richiedere più tempo perché il browser deve scaricare il motore e la lingua italiana.

Carica tutti i file dello ZIP nel repository GitHub mantenendo i nomi esatti.
